# Soccer Betting App

Simple Flask app for placing fixed-amount bets ($5 to $50).

## Setup (macOS)

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Run

```bash
python3 app.py
```

Open http://127.0.0.1:5000

## Notes
- Bets allowed: 5, 10, 15, ..., 50
- This is a demo app; no real payments are processed.
