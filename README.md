# Soccer Betting App

Simple Flask app for placing fixed-amount bets ($5 to $50).

## Setup

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## Run

```powershell
python app.py
```

Open http://127.0.0.1:5000

## Notes
- Bets allowed: 5, 10, 15, ..., 50
- This is a demo app; no real payments are processed.
